### 🎯 Unit Goals


Welcome to Unit 5! In this unit, you will master the basics of object-oriented programming (OOP) with Python through hands-on practice with classes, instances, methods, and attributes. You'll learn how to model real-world entities in your code using classes and to manipulate these entities using methods. This unit will guide you through various scenarios from creating your own data types like Pokémon and linked lists to manipulating objects and their interactions.


By the end of this unit, you will be able to:


- Define and use classes and objects

- Use class attributes and methods to manage object data

- Create and manipulate a singly linked list

- Solve problems requiring traversal of a singly linked list

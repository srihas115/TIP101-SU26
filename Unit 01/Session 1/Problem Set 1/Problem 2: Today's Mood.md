# Problem 2: Today's Mood

The following function uses a variable, `mood` to print out `"Today's mood: 😎"`. Copy this code into your IDE and update the `mood` variable to print out your mood for today.


```python
def todays_mood():
    mood = "😎"
    print("Today's mood: " + mood)

todays_mood()
```

Example Output: `Today's mood: 🥱`


### 💡 Hint: Variables


A variable in Python stores data with a descriptive name, enhancing the clarity and readability of our programs for both the author and others. Variables are created when you first assign them a value and can be modified later.


Python employs dynamic typing, meaning the type of a variable doesn't have to be explicitly stated upon creation; instead, the type is inferred at runtime based on the assigned value.


```python
my_name = "John Smith"
my_age = 21
```

The `""` around the text indicates to Python the variable is a string, and the 21 is an integer as no decimal places are provided.

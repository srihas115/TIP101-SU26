### 🎯 Unit Goals


Welcome to Unit 9! In this unit you'll continue to advance your problem solving skills with binary trees. You'll practice more depth first search traversal problems and learn a new type of traversal: breadth first search.


By the end of this unit, you will be able to:


- Perform a breadth first search or level order traversal of a binary tree

- Articulate the differences between a breadth first and depth first search

- Solve problems that require traversing and manipulating nodes in a tree
